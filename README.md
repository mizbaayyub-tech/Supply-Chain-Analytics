# Supply-Chain-Analytics
**# 🚚 Supply Chain Analytics Project

## 📌 Project Overview

This is an end-to-end Supply Chain Analytics project focused on analyzing inventory, pricing, stock levels, lead times, order quantities, and customer demographics using Excel, SQL, Python, and Power BI.

## 🎯 Objectives

- Clean and prepare raw supply chain data
- Analyze inventory and revenue
- Perform SQL-based business analysis
- Conduct exploratory data analysis using Python
- Build an interactive Power BI dashboard

## 🛠 Tools Used

- Microsoft Excel
- SQL Server
- Python
- Pandas
- Plotly
- Power BI
- Git & GitHub

## 📂 Project Progress
